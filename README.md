# Pro Passport Photo Maker Nepal — Cyber Studio Edition

Frontend-only passport photo studio with:
- Nepal-ready size presets
- Browser background removal using BodyPix
- Crop/drag/zoom face alignment
- A4 sheet generator
- PDF / PNG export
- Direct print
- Undo / redo
- Local JSON save / load

## Run
Open `index.html` in a modern browser.

## Notes
The app uses CDN-hosted libraries for Cropper.js, Fabric.js, jsPDF, TensorFlow.js, and BodyPix.
